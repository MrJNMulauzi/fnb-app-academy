# FNB App Academy Certificate

This repository contains my Certificate of Completion from the FNB App Academy programme, completed in partnership with IT Varsity.

## Programme Overview

The FNB App Academy is a practical technology programme focused on full-stack web and app development fundamentals, problem-solving, and digital innovation.

## Skills & Technologies Covered

- HTML
- CSS
- JavaScript
- Python Fundamentals
- API Concepts
- Backend Basics
- Database Fundamentals
- UI/UX Principles
- User-Centered Design
- App Development Concepts
- Digital Business Solutions

## Key Learning Outcomes

- Developed understanding of modern web and app development workflows
- Built foundational programming and frontend development skills
- Worked with APIs and digital solution concepts
- Applied problem-solving and analytical thinking in practical scenarios
- Strengthened collaboration and project-based learning skills
